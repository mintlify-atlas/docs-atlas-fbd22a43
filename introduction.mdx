## What is File Viewers?

File Viewers is a lightweight, cross-platform desktop application built with Tauri 2 that provides a unified interface for viewing and editing **Markdown**, **JSON**, and **CSV** files. It combines a Monaco-powered code editor with format-specific preview panels, offering a seamless development and data exploration experience.

## Why File Viewers?

Whether you're writing documentation, inspecting API responses, or analyzing datasets, File Viewers provides specialized tools for each format:

- **Live preview** as you type (Markdown and JSON)
- **GitHub Flavored Markdown** rendering with syntax-highlighted code blocks
- **Interactive JSON tree viewer** with expand/collapse controls
- **Advanced CSV table** with sorting, filtering, and SQL query support
- **Native OS integration** via drag-and-drop and keyboard shortcuts (⌘O/Ctrl+O)

### CSV Viewer Highlights

The CSV mode includes production-grade features:

- **SQL query mode**: Run `WHERE` conditions against your CSV data using in-memory SQL (powered by alasql)
- **Search with debouncing**: Global filter with 300ms debounce for smooth performance
- **Column management**: Resize columns, sort by clicking headers (unsorted → asc → desc)
- **Status bar**: Shows row/column count, selected cell coordinates, encoding (UTF-8), and line endings (LF/CRLF)

<Note>
SQL mode registers your CSV data as a table named `csv`. Type conditions like `age > 25 AND city = 'NYC'` and press ⌘↵ to execute.
</Note>

## Tech Stack

File Viewers is built with modern web and native technologies:

| Layer | Technology |
|-------|------------|
| App shell | Tauri 2 |
| UI framework | React 19 + TypeScript |
| Build tooling | Vite 7 + Tailwind CSS 4 |
| Code editor | Monaco Editor (local bundle) |
| Markdown | react-markdown + remark-gfm + rehype-highlight |
| JSON viewer | react-json-view-lite |
| CSV table | TanStack Table v8 + PapaParse |
| CSV SQL | alasql (in-memory SQL engine) |
| UI primitives | Base UI (`@base-ui/react`) |
